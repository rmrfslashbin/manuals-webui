About Browser Storage

Your API configuration is stored in your browser's sessionStorage. This means:

  • Settings cleared when you close the tab/window
  • More secure than localStorage (shorter exposure)
  • You'll need to reconfigure each browser session
  • Settings are device-specific (not synced)

Security Considerations

Browser sessionStorage is not encrypted. Your API key is stored in plain text (but only for this tab session) and can be accessed by:

  • Browser DevTools (inspect element)
  • Browser extensions
  • Malicious scripts (XSS attacks)

Only use this on trusted devices and networks.
